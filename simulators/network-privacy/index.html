Why Latency Matters

  • Gaming: Low latency (under 50ms) is essential for real-time games
  • Video Calls: High latency causes delays and awkward pauses
  • Web Browsing: Lower latency means pages load faster
  • Distance: Physical distance affects latency - servers closer to you = lower latency

How Encryption Works

  1. Plain Text: Your original message (readable)
  2. Encryption Key: A secret code used to scramble the message
  3. Cipher Text: The scrambled, unreadable message
  4. Decryption: Using the key to convert cipher text back to plain text
